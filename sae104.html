SAÉ 1.04 · Semestre 1

Se présenter sur internet

Concevoir et déployer un site web personnel, maîtriser les technologies du web et construire une identité numérique professionnelle.

Gwendal Cauchon Compétence RT3 – Créer des outils et applications AC13.01 · AC13.04

Description & Objectifs

Cette SAÉ vise à construire une présence professionnelle sur internet à travers la création d'un site web personnel. Elle mobilise des compétences en développement web (HTML/CSS), en architecture de site et en expression de soi dans un contexte numérique professionnel.

  • Concevoir l'architecture d'un site web personnel clair et navigable.
  • Implémenter des pages web en HTML5 et CSS3 (ou avec un framework comme Tailwind CSS).
  • Publier le site sur un hébergement accessible publiquement.
  • Adopter une charte graphique cohérente et une identité visuelle professionnelle.
  • Valoriser son parcours, ses compétences et ses projets de manière structurée.
HTML5 CSS3 Tailwind CSS Font Awesome JavaScript

Apprentissages Critiques (AC)

AC13.01 – RT3Utiliser un système informatique et ses outils : éditeur de code, navigateur, outils de développement, hébergement web.
AC13.04 – RT3Connaître l'architecture et les technologies d'un site web : structure HTML, styles CSS, interactions JavaScript, déploiement.

Tâches réalisées

  • Conception de l'architecture : Définition des pages du site (accueil, projets/SAÉ, présentation, bilan), création d'une arborescence claire et d'une navigation intuitive.
  • Développement frontend : Codage en HTML5 et CSS3 avec Tailwind CSS pour le responsive design, intégration des icônes Font Awesome, développement d'interactions JavaScript (toggle, navigation).
  • Charte graphique : Choix d'une palette de couleurs cohérente (bleu, blanc, gris), typographie lisible, mise en page épurée.
  • Déploiement : Mise en ligne du site sur un hébergement accessible, vérification de la compatibilité multi-navigateurs.
  • Contenu : Rédaction des textes de présentation personnelle, description des SAÉ et des compétences développées.

Résultats obtenus

Le site portfolio est opérationnel et accessible en ligne. Il présente l'ensemble des SAÉ du semestre 1 avec une structure homogène. La charte graphique bleue a été maintenue sur toutes les pages, garantissant une cohérence visuelle. Le menu de navigation permet un accès rapide à toutes les rubriques.

Autoévaluation

Cette SAÉ est celle qui m'a le plus impliqué personnellement, car elle porte directement sur ma propre image professionnelle. J'ai apprécié le défi de combiner esthétique et contenu technique. La principale difficulté a été de trouver le bon niveau de détail dans la présentation des SAÉ : ni trop synthétique, ni trop exhaustif.

Je maîtrise bien l'AC13.04 (architecture web et technologies), et je progresse sur l'AC13.01 (maîtrise des outils). Je souhaite approfondir le JavaScript pour rendre le site plus interactif.


Problème rencontré Solution envisagée ou envisageable
Responsive design difficile à gérer sur mobile pour certaines sections. Tester systématiquement sur plusieurs tailles d'écran et utiliser les utilitaires responsive de Tailwind.
Contenu des pages SAÉ trop long, rendant la lecture fastidieuse. Ajouter des systèmes d'accordéon (toggle) pour masquer les détails par défaut.
Hébergement et déploiement : gestion des chemins relatifs pour les images et fichiers. Organiser les fichiers en dossiers thématiques (img/, pdf/, etc.) et vérifier chaque lien après déploiement.
